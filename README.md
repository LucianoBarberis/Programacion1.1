## 💻 Cómo Compilar los Programas
1. Asegúrate de tener GCC instalado
2. Navega hasta la carpeta del ejercicio
3. Compila usando: `gcc nombreArchivo.c -o nombreArchivo.exe`
4. Ejecuta el programa: `nombreArchivo.exe`
